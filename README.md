# 文件损坏器

一个简单的前端文件损坏工具，用于随机损坏现有文件或生成指定大小的损坏文件。

## 功能特性

- **损坏现有文件**：上传任意文件，进行随机字节损坏处理
- **生成损坏文件**：创建指定大小和格式的随机损坏文件
- **自定义文件名**：生成文件时可自定义文件名
- **拖拽上传**：支持拖拽文件到页面进行上传
- **进度显示**：操作过程中显示进度提示
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- **前端框架**：原生HTML/CSS/JavaScript
- **样式**：Tailwind CSS
- **动画**：GSAP (GreenSock Animation Platform)
- **字体**：Google Fonts (Inter)

## 运行项目

由于这是纯前端项目，无需服务器：

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html` 文件
3. 开始使用

## 文件结构

```
├── index.html      # 主页面
├── script.js       # JavaScript 逻辑
└── style.css       # 样式文件
```
